# CarLens Frontend

CarLens is an AI-powered used car marketplace built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Users can browse cars, compare vehicles, save favorites, and get AI-powered reviews and recommendations.

## Features

* User Login & Registration
* JWT Authentication
* Browse Used Cars
* Car Details Page
* AI Car Review & Deal Score
* AI Car Comparison
* Favorite Cars
* Compare up to 3 Cars
* User Profile
* Responsive Design
* Admin Portal

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Zustand
* Axios
* Lucide React

## Installation

Clone the repository:

```bash
git clone https://github.com/ARJUNaju0/carlens-frontend.git
```

Go to the project folder:

```bash
cd carlens
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000/api
NEXT_PUBLIC_BACKEND_URL=http://127.0.0.1:8000
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

## Build

```bash
npm run build
npm start
```
